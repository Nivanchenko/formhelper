# Начало работы

## Подключение библиотеки

1. Скачать последний скомпилированный файл расширения из раздела [releases](/formhelper/releases) Github или собрать самостоятельно из исходников
2. Установить расширение в базу 1С Предприятия
3. Создать своё расширение, которое будет содержать ваши изменения в конфигурации
4. Заимствовать форму из основной конфигурации в созданное расширение
5. В процедуре "ПриСозданииНаСервере" реализовать неоходимый функционал. Например:

```bsl
&НаСервере
Процедура РСШ_ПриСозданииНаСервереПосле(Отказ, СтандартнаяОбработка)

    РаботаСФормамиСервер.НовыйПостроительФорм(ЭтаФорма)
        .РеквизитСтрока("РеквизитСтрока", 150)
        .ПолеВвода("РеквизитСтрока")
            .Заголовок("Очень полезный реквизит")
            .ПутьКДанным("РеквизитСтрока")
        .Применить()
    ;

КонецПроцедуры
```

6. Итог: вы прекрасны! У вас получился код который легко поддерживать при очередном обновлении конфигурации.
